# Способ 1. Создание страницы из выбранного раздела

**Рекомендуется использовать данный способ**, поскольку он привязывает страницу к нужному разделу и снижает риск её "потери" среди непривязанных страниц.
1. Определить раздел, в котором должна быть размещена инструкция.
2. Открыть страницу этого раздела и нажать кнопку "Править страницу".
3. В режиме редактирования доьавить ссылку на новую страницу по аналогии с существующими ссылками.

   Пример разметки:

   `[[group:instructions:common_instructions|Общая инструкция]]`

   Где: 
   - `group:instructions:common_instructions` - адрес страницы,
   - `Общая инструкция` - отображаемое название страницы.
4. Сохранить изменения, нажав кнопку "Сохранить". На странице раздела появится ссылка на новую страницу.
5. Перейти по появившейся (обычно выделенной красным) ссылке.
6. На открывшейся страницу нажать кнопку "Создать страницу".
