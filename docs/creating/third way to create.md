# Способ 3. Создание страницы через URL
1. Указать название новой страницы в URL после базового адреса.

     Пример: 

    `http://wiki.example.com/main/Новая_страница`

2. После перехода по этому адресу система сообщит, что страницы не существует.
3. Нажать кнопку "Создать страницу" и перейти к её редактированию.
