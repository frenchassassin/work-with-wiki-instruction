# Удаление страницы
Для удаления страницы:
1. Нажать кнопку "Править страницу".
2. В режиме редактирования удалить всё содержимое страницы.
3. Нажать кнопку "Сохранить".

При сохранении пустой страницы вики-движок удаляет саму страницу и её имя из пространства страниц.

> *Примечание*. Старые версии удалённой страницы, как правило, сохраняются в специальном хранилище версий ("чердак"), откуда их можно при неоьходимости восстановить.
>
> Пример справочной страницы: `https://dokuwiki.example.com/attic`
