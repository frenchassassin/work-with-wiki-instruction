# Редактирование страницы
Для редактирования существующей страницы используется кнопка **"Править страницу"**, расположенная на самой странице.

При создании или редактировании открывается окно редактирования, в котором вводится вики-разметка.

Чтобы посмотреть, как будет выглядеть страница до окончательного сохранения, используется кнопка **"Просмотр"**.

После завершения редактирования необходимо нажать кнопку **"Сохранить"**. Страница будет преобразована в HTML, используемый как основной формат отображения.

Содержимое страницы при необходимости может быть экспортировано в другие форматы.
